Gestor de Inventario

Aplicación web full stack para la gestión eficiente de inventarios, desarrollada con .NET MVC, Entity Framework, SQL Server, Bootstrap, JavaScript y CSS3. Incluye:

  • Operaciones CRUD para productos y pedidos.
  • Autenticación JWT (simétrica y asimétrica).
  • Integración con PayPal para pagos y suscripciones.
  • Generación de informes PDF y notificaciones por email.
  • Panel de administración con roles y gestión de usuarios.
  • Soporte para Docker y Redis.
