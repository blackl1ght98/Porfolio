Glucontrol

Glucontrol es una aplicación web diseñada para personas diabéticas, desarrollada en equipo para facilitar el seguimiento y gestión de mediciones de glucosa de manera accesible. El frontend utiliza Angular, HTML, CSS3 y TypeScript, mientras que el backend se desarrolló con Entity Framework, C# y una base de datos.

Descripción del Proyecto

Participé en un proyecto en grupo dividido en dos equipos: frontend y backend. Junto con otro compañero, me encargué del backend, mientras que el resto del equipo desarrolló el frontend. Aunque las tareas estaban divididas, colaboramos activamente, apoyándonos mutuamente en ambas áreas según fuera necesario.

  • Backend: Desarrollado con Entity Framework y C#, con una base de datos para gestionar mediciones y datos de usuarios.
  • Frontend: Construido con Angular, HTML, CSS3 y TypeScript para una interfaz dinámica y accesible.
  • Funcionalidades Clave:
    • Descarga de mediciones en formato PDF.
    • Envío de mediciones por correo electrónico.
