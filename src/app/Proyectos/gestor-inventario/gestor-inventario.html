Gestor de Inventario - E-commerce

Gestor de Inventario es una solución avanzada para la gestión de inventarios en plataformas de comercio electrónico, desarrollada con .NET 9.0 MVC para el frontend y backend, garantizando robustez, escalabilidad y seguridad.

Tecnologías Utilizadas

.NET 9.0 MVC · C# · Entity Framework Core · SQL Server · Redis · Docker · Bootstrap · JWT (simétrico y asimétrico) · PayPal API · AutoMapper · Polly

Novedades

Actualizado a .NET 9.0, este proyecto ha evolucionado de un gestor de inventario básico a una completa plataforma de e-commerce con funcionalidades avanzadas.

Características del Proyecto

  • Gestión de Datos: Operaciones CRUD para usuarios, proveedores, productos, pedidos y su historial.
  • Autenticación Robusta: Soporte para autenticación con tokens JWT, incluyendo:
    • Simétrica.
    • Asimétrica con claves fija.
    • Asimétrica con claves dinámicas.
  • Generación de Informes: Exportación de reportes en PDF del historial de pedidos y productos.
  • Notificaciones por Email: Alertas automáticas para stock bajo.
  • Registro y Acceso: Registro de usuarios con confirmación por correo electrónico.
  • Panel de Administración: Gestión de cuentas de usuario con interfaz intuitiva.
  • Sistema de Roles: Control de acceso basado en permisos por niveles.
  • Pasarela de Pago: Integración con PayPal para pagos seguros.
  • Restablecimiento de Contraseña: Recuperación de cuentas mediante correo con clave temporal.
  • Flexibilidad en Autenticación: Alternancia entre modos de autenticación vía código.
  • Gestión de Usuarios: Alta y baja de usuarios, individual o masiva, por administradores.
  • Contenedores y Caché: Soporte para Docker y Redis para despliegue y rendimiento optimizado.
  • Reembolsos: Funcionalidad para procesar reembolsos desde el panel de administración.
