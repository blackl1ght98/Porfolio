Gestor Inventario - E-commerce

Gestor Inventario es una aplicación diseñada para la gestión eficiente de inventarios en plataformas de comercio electrónico. El sistema está desarrollado con .NET MVC en el frontend y .NET 9.0 en el backend, ofreciendo una solución robusta, escalable y segura.

NOVEDADES

Este proyecto ha sido actualizado a la última versión de .NET (9.0). Ha evolucionado de un simple gestor de inventario a una completa solución de comercio electrónico.

Características del Proyecto

  • Gestión de Datos: Operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para usuarios, proveedores, productos, pedidos y su historial.
  • Autenticación Robusta: Soporte para autenticación mediante generación de tokens, incluyendo:
    • Autenticación simétrica.
    • Autenticación asimétrica con claves pública y privada fijas.
    • Autenticación asimétrica con claves pública y privada dinámicas.
  • Generación de Informes: Exportación de reportes en formato PDF del historial de pedidos y productos.
  • Notificaciones por Correo Electrónico: Alerta automática cuando el stock de un producto es bajo.
  • Registro y Acceso de Usuarios: Registro de nuevos usuarios con confirmación por correo electrónico.
  • Panel de Administración: Herramienta para gestionar cuentas de usuario.
  • Sistema Basado en Roles: Control de acceso por niveles según permisos asignados.
  • Pasarela de Pago: Integración con PayPal para procesar pagos de manera segura.
  • Restablecimiento de Contraseña: Opción para recuperar la contraseña mediante correo electrónico con una clave temporal.
  • Flexibilidad en la Autenticación: Posibilidad de alternar entre modos de autenticación modificando el código.
  • Gestión de Usuarios: Alta y baja de usuarios, de forma individual o masiva, por parte del administrador.
  • Docker: Configuración lista para desplegar en contenedores Docker.
  • Redis: Integración con Redis para mejorar el rendimiento y la caché de datos.
  • Función de Reembolso: Funcionalidad para procesar reembolsos de pedidos desde el panel de administración.
