Gestor de Tareas

Gestor de Tareas es una aplicación web que permite a los usuarios registrar, organizar y realizar seguimiento de sus tareas pendientes de forma sencilla y eficiente.

Este proyecto fue desarrollado con Angular, Bootstrap, HTML5, JavaScript, CSS3 y Firebase, ofreciendo una experiencia moderna y funcional para la gestión diaria de tareas.

Finalizado en agosto de 2023, el sistema ha demostrado ser útil para mejorar la organización personal de los usuarios, evitando que olviden tareas importantes gracias a su interfaz amigable y sincronización en la nube.

El backend está implementado completamente con Firebase, incluyendo autenticación, base de datos y hosting.
