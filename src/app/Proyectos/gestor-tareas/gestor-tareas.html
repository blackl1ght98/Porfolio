Gestor de Tareas

Gestor de Tareas es una aplicación web para registrar, organizar y seguir tareas pendientes de forma sencilla y eficiente, desarrollada con Angular, Bootstrap, HTML5, JavaScript, CSS3 y Firebase.

Descripción del Proyecto

Finalizado en agosto de 2023, este proyecto ofrece una interfaz moderna y funcional para mejorar la organización personal, evitando olvidos de tareas importantes mediante sincronización en la nube con Firebase.

  • Frontend: Desarrollado con Angular, Bootstrap, HTML5, JavaScript y CSS3 para una experiencia de usuario amigable y dinámica.
  • Backend: Implementado con Firebase, incluyendo autenticación, base de datos en tiempo real y hosting.
  • Funcionalidades: Registro y gestión de tareas, sincronización en la nube, autenticación de usuarios.
