Gestor de Citas

Gestor de Citas es una aplicación para gestionar citas de manera eficiente, con un frontend dinámico en Angular y un backend robusto en .NET 7.0.

Frontend

Construido con Angular, el frontend ofrece una interfaz rica y dinámica para una experiencia de usuario fluida.

  • Registro de Usuario: Creación de cuentas con detalles personalizados.
  • Inicio de Sesión: Acceso seguro con credenciales de usuario.
  • Recuperación de Contraseña: Restablecimiento mediante correo electrónico.
  • Cambio de Contraseña: Actualización de contraseñas desde el perfil.
  • Gestión de Citas: Creación y eliminación de citas según necesidades.

Backend

Desarrollado con .NET 7.0, el backend garantiza alto rendimiento, escalabilidad y seguridad.

  • Autenticación de Usuario: Gestión segura de inicio de sesión y registro.
  • Gestión de Usuarios: Eliminación y recuperación de datos de usuarios.
  • Generación de Tokens: Tokens para autenticación y sesiones seguras.
  • Hashing de Contraseñas: Cifrado de contraseñas para máxima seguridad.
