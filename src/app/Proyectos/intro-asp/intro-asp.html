Intro ASP

Intro ASP es una aplicación web para gestionar cervezas, desarrollada con .NET 7.0 MVC para el frontend y backend, ofreciendo una interfaz dinámica y un sistema robusto y escalable.

Frontend

Construido con .NET MVC, el frontend proporciona una interfaz rica y dinámica para una experiencia de usuario eficiente.

  • Registro de Usuario: Creación de cuentas con detalles personalizados.
  • Inicio de Sesión: Acceso seguro con credenciales de usuario.
  • Restablecimiento de Contraseña: Solicitud al administrador para restablecer contraseñas.
  • Cambio de Contraseña: Actualización de contraseñas desde el perfil.
  • Gestión de Cervezas: Creación y eliminación de cervezas según necesidades.

Backend

Desarrollado con .NET 7.0, el backend garantiza alto rendimiento, escalabilidad y seguridad.

  • Autenticación de Usuario: Gestión segura de inicio de sesión y registro.
  • Gestión de Usuarios: Asignación de roles para control de permisos.
  • Autenticación Basada en Cookies: Gestión de sesiones con estado almacenado en el servidor.
  • Hashing de Contraseñas: Cifrado de contraseñas para máxima seguridad.
