Intro ASP

Frontend El frontend de la aplicación está desarrollado con el popular framework .NET MVC . Este proporciona una interfaz de usuario rica y dinámica, permitiendo a los usuarios interactuar con la aplicación de manera eficiente. Las características clave del frontend incluyen:

  • Registro de usuario: Los nuevos usuarios pueden crear una cuenta proporcionando los detalles necesarios.
  • Inicio de sesión: Los usuarios registrados pueden iniciar sesión en la aplicación utilizando sus credenciales.
  • Restablecimiento de contraseña: Si un usuario olvida su contraseña, puede solicitar al administrador que le restablezca la contraseña
  • Cambio de contraseña: Los usuarios pueden cambiar su contraseña después de iniciar sesión en la aplicación.
    • Gestión de cervezas: Los usuarios pueden agregar y eliminar cervezas según sus necesidades.

    Backend El backend de la aplicación está construido con .NET 7.0, un marco de trabajo robusto y escalable que permite el desarrollo de aplicaciones de alto rendimiento. Las funcionalidades principales del backend incluyen:

  • Autenticación de usuario: El backend maneja el inicio de sesión y el registro de los usuarios, asegurando que solo los usuarios autorizados puedan acceder a la aplicación.
  • Gestión de usuarios: Los usuarios tienen roles asignados y pueden hacer lo que le permita ese rol
  • Autenticación en base a cookies: Para mantener las sesiones de los usuarios y permitir la autenticación de los usuarios la autenticación basada en cookies tiene un estado que se almacena en el servidor este estado dice si el usuario a iniciado sesion o a cerrado sesion
  • Hashing de contraseñas: Para garantizar la seguridad de las contraseñas de los usuarios, estas se cifran utilizando un algoritmo de hashing.
