Desarrollando soluciones innovadoras en el universo digital

Soy Guillermo Fuentes Buenosvinos, un profesional apasionado por la tecnología con experiencia en desarrollo web, seguridad informática y administración de sistemas. Actualmente, curso el programa de Técnico Superior en Desarrollo de Aplicaciones Web, fortaleciendo mis competencias para crear soluciones digitales eficientes, escalables y seguras.

Me destaco en la resolución de problemas complejos, la optimización de procesos y la enseñanza de tecnologías. Mi proyecto más relevante es el Gestor de Inventario, una aplicación web desarrollada con .NET MVC, Entity Framework, Bootstrap, JavaScript y CSS3, diseñada para gestionar inventarios con funcionalidades avanzadas como autenticación robusta, pagos con PayPal y generación de informes.

Domino tecnologías como C#, .NET, SQL Server, MongoDB, React, Angular y Node.js, con un enfoque especial en el ecosistema .NET (MVC, Web API, Blazor). Estoy comprometido con la exploración continua de nuevas herramientas para ofrecer soluciones innovadoras y de alto impacto.

Proyectos Destacados

  • Gestor de Inventario
    Aplicación web full stack para la gestión eficiente de inventarios, desarrollada con .NET MVC, Entity Framework, SQL Server, Bootstrap, JavaScript y CSS3. Integra funcionalidades avanzadas como:

    • Autenticación basada en JWT con modos simétrico y asimétrico.
    • Pasarela de pago con PayPal y gestión de suscripciones.
    • Generación de informes en PDF y notificaciones por correo electrónico.
    • Panel de administración con sistema de roles y gestión de usuarios.
    • Soporte para Docker y Redis en entornos de desarrollo.
